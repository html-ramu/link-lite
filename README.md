# 🔗 Link Lite

A simple, minimal URL shortener built with HTML, CSS, and vanilla JavaScript.
Hosted on GitHub Pages. No backend, no database, no frameworks.

## 🌐 Live Site

[https://html-ramu.github.io/link-lite/](https://html-ramu.github.io/link-lite/)

## ✨ Features

- Shorten any long URL instantly
- Dark mode / Light mode toggle
- Recent links history (saved in browser)
- Minimal black & white design
- Fully mobile responsive
- 100% free — hosted on GitHub Pages

## 🛠️ Built With

- HTML
- CSS (CSS Variables for theming)
- Vanilla JavaScript
- GitHub Pages (hosting)
